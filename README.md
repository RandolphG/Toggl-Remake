## Toggle Remake

### Installation process :

- clone repo
- navigate to project folder
- install all dependencies with npm
- run with npm


```bash
$ git clone https://github.com/RandolphG/toggl-v1.git
$ cd project
$ npm i
$ npm run electron
```
